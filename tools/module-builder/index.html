Entitas & Aksi

Contoh: orders, order_items, payments
getAll()
List semua record dengan filter org
getById()
Ambil satu record by ID
create()
Buat record baru + trigger integrasi
update()
Update record + revalidate cache
delete()
Hapus record (soft delete jika ada deleted_at)
updateStatus()
Ubah status + record finansial jika PAID
Tekan Enter untuk tambah status
Konfigurasi Chart of Accounts
Tentukan akun debit/kredit yang dipakai saat transaksi dicatat ke General Ledger.
Integrasi Financial tidak dipilih
Konfigurasi CoA tidak diperlukan untuk modul ini.
Pola Jurnal: Debit akun penerima (Kas/Piutang) → Kredit akun sumber (Pendapatan/Hutang).
Untuk beban: Debit Beban → Kredit Kas.
