Data apa yang disimpan?
Sebutkan nama-nama "tabel" — yaitu jenis data yang perlu disimpan oleh modul ini. Kalau bingung, lihat contoh di bawah.
Contoh: Modul Workshop → tabelnya: workshop_orders (daftar servis), workshop_items (sparepart yang dipakai). Cukup tulis nama dalam bahasa Indonesia pun bisa, nanti otomatis dikonversi.
Biasanya cukup 1–3 tabel. Minimal 1.
Contoh: MENUNGGU, DIPROSES, SELESAI, DIBAYAR — ketik lalu tekan Enter

Cara install ke project

Pilih cara yang paling mudah buat kamu:

Download Langsung

Klik tombol ini → pilih folder nizam-app di komputermu → semua file masuk otomatis ke tempat yang benar.

Pakai Terminal

Copy script ini → buka terminal → masuk ke folder nizam-app → paste → Enter. Selesai.

Setelah file masuk ke project, lakukan ini:
1. Buka terminal di folder nizam-app
2. Jalankan: npx supabase db push — untuk buat tabelnya di database
3. Jalankan: npm run dev — untuk cek tidak ada error
4. Buka aplikasi dan coba tambah data pertama
