title: Reference Library
description: Kumpulan dokumen sumber yang disinkronkan dari markdown lama di root folder docs.
---

Section ini berisi dokumen sumber yang sudah ada sebelumnya di folder `/docs`. File-file tersebut disinkronkan otomatis ke sini setiap kali Anda menjalankan `npm run sync-content`, `npm run dev`, atau `npm run build`.

Gunakan section ini untuk:

- menjaga jejak dokumentasi lama,
- membuka referensi teknis dengan cepat,
- mengisi bahan awal saat menulis halaman docs yang lebih terkurasi.

> Untuk dokumen yang lebih ramah navigasi, mulai dari halaman [Getting Started](/guides/getting-started/).
